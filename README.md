## Dependency Installation:
1. `npm install`
2. `npm install express`
3. `npm install handlebars`
4. `npm install express-handlebars`
5. `npm install -g nodemon`

## To Run with `nodemon`
`npm run dev`
 -  This auto restarts the application when there's new changes
